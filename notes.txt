aws ec2 describe-key-pairs --key-names key-pair-name

aws ec2 describe-key-pairs --key-names public

Retrive public key material from the private key:

ssh-keygen -y -f /path_to_key_pair/my-key-pair.pem
>>> ssh-keygen -y -f ~/.ssh/public.pem

Making .pem file secure

IN wondows right click file
properties > security > advanced
Disable inheritance (make explicit)
Remove all permissions apart from emmanora@hotmail.com

Save file in .ssh folder in home: ~/.ssh

sudo chmod 400 filename.pem

To SSH into instance

ssh -i /path/key-pair-name.pem instance-user-name@instance-public-dns-name

For a Ubuntu AMI, the user name is ubuntu


sudo ssh -i ~/.ssh/public.pem ubuntu@ec2-18-175-137-28.eu-west-2.compute.amazonaws.com
sudo ssh -i ~/.ssh/public.pem ubuntu@35.178.210.157
